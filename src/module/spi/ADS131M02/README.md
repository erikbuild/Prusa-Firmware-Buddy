# ADS131M02

This is a custom controller for SPI ADC converter ADS131M02 by Texas Instruments.

* [Datasheet](https://www.ti.com/lit/ds/symlink/ads131m02.pdf)
* 2 separately configurable channels
* Requires external MCOS clock source (2 MHz - 8 MHz)
* SPI Configuration:
  * CPOL=0
  * CPHA=1
  * Max Clock Frequency: 10 MHz
